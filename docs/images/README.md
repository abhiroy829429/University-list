# Documentation Images

This folder contains images specifically for documentation files.

## Usage

Images in this folder can be referenced from documentation files:

```markdown
![Deployment Guide](images/deployment-screenshot.png)
![Setup Steps](images/setup-diagram.png)
```

## Recommended Images for Docs

- **Screenshots** of deployment process
- **Diagrams** explaining architecture
- **Step-by-step** guides with images
- **Platform comparison** charts
